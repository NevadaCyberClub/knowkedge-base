The command that can be used here is
>  hashcat -m 0 -a 3 test.hash UNRCyber{?d?d?d?d}
?d signifies to replace ?d with a number. when ran, this will go through
all possible combinations of UNRCyber{####}.

for more info on hashcat run the command:
> hashcat --help
